# Calculadora de presupuestos
Este es mi primer proyecto en Python. Es una herramienta diseñada para gestionar y calcular presupuestos de forma rapida (creando un PDF con la informacion en el proceso)

## Caracteristicas
* Intefas Sencilla
* Calculo automatico
* Ideal para calcular presupuestos

## ¿Como utilizar?
Solo necesitas ejecutar el archivo llamado "Proyecto01.py" en tu entorno de Python

## Deberias tomar en cuenta que este proyecto todavia no esta terminado, podria contener algunos errores como: No poder poner Dias,Horas,Meses en el calculo, si haces esto solo pon los numero sin nada mas, si haces caso omiso podria darte error,
